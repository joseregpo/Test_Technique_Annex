  	Pour l'installation du projet il faut initialiser la base des données rélationnelle avec mySQL en important le fichier placé dans le 
    	Dossier Database
	
	Il faut également initialisar la variable URL (dans js/main.js) avec l'url où se trouve l'API dans le serveur où le projet est installé,
	par exemple "127.0.0.1/php/api/read.php"

	Puis dans php/config/Database.php sont placés les variables de développement pour accéder à la base de données

   	Le projet fonctionne avec php version 7.4.*
